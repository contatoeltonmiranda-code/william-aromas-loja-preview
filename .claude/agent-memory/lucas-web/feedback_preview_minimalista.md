Preview inicial de copy (antes de design aprovado) deve ser documento minimalista de leitura — nao site estilizado.

**Why:** Elton precisa ler/revisar a copy organizada por pagina. Site estilizado (paleta, hero, cards, gradientes, fontes serif) distrai da revisao de texto. Em 2026-05-05 entreguei preview Aromas do Deserto estilizado e foi rejeitado.

**How to apply:**
- CSS minimo: system-ui, max-width 720px, branco (#fff), preto (#111), links azul (#0066cc)
- Sem cards, grids coloridos, hero com fundo, gradientes, sombras, animacoes, hover effects
- Nav simples como lista de links no topo
- Heros viram H1 + paragrafo + CTA texto
- Tabelas HTML simples (border 1px #ddd)
- CTAs sao links sublinhados, nao botoes
- Listas/dl para produtos, nao cards
- Footer 1 linha discreta
- So criar versao estilizada DEPOIS da copy aprovada
